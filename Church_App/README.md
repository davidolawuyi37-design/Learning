# Church Member Management App

A simple Python console application for managing church members.

## Features

- Add new members with name, age, contact, and membership date
- List all members
- Find a member by name
- Data persists in a JSON file

## How to Run

1. Ensure Python is installed.
2. Run `python church_app.py`
3. Follow the menu options.

## Requirements

- Python 3.x
- No external libraries needed (uses built-in json and os)
